Esta herramienta permite:

Crear libros digitales completos, automáticamente: eliges temas, idioma, estilo visual, número de capítulos, etc., y la IA se encarga de escribir, ilustrar y maquetar.
Elegir entre varios modelos de IA para generar texto (por ejemplo, GPT-4.1, Gemini 2.5, entre otros) y diferentes modelos para generar imágenes, entre ellos Nano banana y Seedream.
Seleccionar estilos artísticos, paletas de colores, estilos visuales muy variados (desde realismo, acuarela, steampunk, cyberpunk, hasta pixel art, estilo cómic, etc.).
